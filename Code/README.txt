HOW TO RUN CODE:

1. Open doorkey.py
2. Under main, please comment in part A (commented in by default) to see results of known map
3. Comment out part A and comment in part B to see the results of random map
4. All results can be viewed via gif from the gif folder - doorkey.gif
5. door_key_problem_random.py holds the code for the DPA of random map, doorkey.py holds the 
   code for the DPA of known map (and the main).
